# Getting Started with MyFirstRepo

This guide will help you get started with MyFirstRepo.

## Prerequisites

- Python 3.6 or higher (for Python examples)
- Node.js (for JavaScript examples)
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Rosellpc/MyFirstRepo.git
cd MyFirstRepo
```

2. You're ready to go! This is a learning repository with example code.

## Running Python Examples

Navigate to the repository root and run:

```bash
# Run the hello example
python src/hello.py

# Run the hello example with the example script
python examples/hello_example.py
```

## Running JavaScript Examples

You can run the JavaScript examples in different ways:

### In a browser:
1. Open the HTML file in a browser
2. Open the browser console to see the output

### In Node.js:
```bash
# First, load the calculator.js in your script, then:
node examples/calculator_example.js
```
